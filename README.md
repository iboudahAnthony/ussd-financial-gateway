# USSD Financial Gateway Core
A modular Python prototype for a decentralized financial gateway, designed using a Work Breakdown Structure (WBS).

## Features
- **State Machine:** Multi-level menu navigation.
- **Persistence:** Local JSON ledger for transaction history.
- **Session Security:** State-guarded input validation.
